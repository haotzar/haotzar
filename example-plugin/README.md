# תוסף דוגמה - מחשבון

זהו תוסף דוגמה להדגמת מערכת התוספים באפליקציה.

## מבנה התוסף

```
example-plugin/
├── manifest.json    # קובץ תצורה עם מידע על התוסף
├── index.html       # קובץ HTML ראשי
├── style.css        # עיצוב התוסף
├── script.js        # לוגיקה של התוסף
├── icon.png         # אייקון התוסף (אופציונלי)
└── README.md        # תיעוד
```

## קובץ manifest.json

קובץ זה מכיל מידע חיוני על התוסף:

- **name**: שם התוסף
- **version**: גרסת התוסף
- **description**: תיאור קצר
- **author**: שם המפתח
- **main**: קובץ HTML ראשי
- **icon**: נתיב לאייקון (אופציונלי)
- **styles**: מערך של קבצי CSS (אופציונלי)
- **scripts**: מערך של קבצי JavaScript (אופציונלי)

## איך להשתמש

1. פתח את האפליקציה
2. לחץ על כפתור "כלים" בהידר
3. בחר "תוספים"
4. לחץ על "הוסף תוסף"
5. בחר את תיקיית `example-plugin`
6. התוסף יותקן ויופיע ברשימה
7. לחץ על התוסף כדי להפעיל אותו

## יצירת תוסף משלך

1. צור תיקייה חדשה לתוסף
2. צור קובץ `manifest.json` עם המידע הנדרש
3. צור קובץ HTML ראשי
4. הוסף קבצי CSS ו-JS לפי הצורך
5. הוסף אייקון (אופציונלי)
6. טען את התוסף דרך מנהל התוספים

## הגבלות אבטחה

התוספים רצים בתוך iframe עם sandbox, מה שמגביל את הגישה שלהם למערכת.
זה מבטיח שתוספים לא יכולים לגשת למידע רגיש או לפגוע באפליקציה הראשית.

## רעיונות לתוספים
